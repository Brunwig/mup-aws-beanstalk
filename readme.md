## mup-aws-beanstalk

Plugin for Meteor Up to deploy using AWS Beanstalk.

Features:
- Load balancing with support for sticky sessions and web sockets
- Autoscaling
- Meteor settings.json
- Zero downtime deploys
- Automatically uses the correct node version
